# **App Name**: PhionixCRM

## Core Features:

- User Authentication: Secure user login and authentication system with role-based access control.
- Customer Management: Create, read, update, and delete customer profiles with detailed information and history.
- Interaction Tracking: Log all interactions with customers, including calls, emails, meetings, and notes.
- Opportunity Management: Manage sales opportunities with stages, assignments, and estimated values.
- Task Management: Create and assign tasks to employees with priorities, deadlines, and statuses.
- AI-Powered Insights: Leverage a tool that uses AI to analyze customer interactions and provide insights, summaries, and action suggestions.
- Reporting and Dashboards: Generate reports and dashboards for sales performance, customer activity, and support metrics.

## Style Guidelines:

- Primary color: HSL (45, 80%, 50%) translates to a vibrant gold (#E6B800) to signify business value.
- Background color: HSL (45, 20%, 95%) translates to a very light pale gold (#F7F6F0) for a professional backdrop.
- Accent color: HSL (75, 60%, 40%) translates to a deep olive green (#669900) for secondary actions and highlights.
- Body font: 'Inter', sans-serif, for a modern and readable text.
- Headline font: 'Space Grotesk', sans-serif, for a modern and readable header.
- Use clean and professional icons to represent different features and actions.
- A clean and structured layout to facilitate easy navigation
